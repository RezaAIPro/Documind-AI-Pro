# Documind AI Pro

AI PDF Assistant for Persian & English Documents

## Features

- Read PDF files
- Chat with your documents
- Completely offline
- Persian support
- Multiple AI models
- Beautiful interface
- Fast responses

## Supported Models

- Gemma 2
- Llama 3.1
- Mistral

## Technologies

- Python
- Gradio
- Ollama
- PyPDF2
